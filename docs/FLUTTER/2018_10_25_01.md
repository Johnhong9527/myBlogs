### 纵向文字列表显示
```dart
// 文字list渲染
_textListView() {
  return Container(
    child: ListView.builder(
      padding: EdgeInsets.all(8.0),
      itemExtent: 20.0,
      itemBuilder: (BuildContext context, int index) {
        return Text('entry $index');
      },
    ),
  );
}
```

## 点击事件
该点击事件,可作用任意元素中
```dart
// 图片点击事件
Widget image_on_tap(info) {
  // print(info[0]);
  // print(93);
  return GestureDetector(
    onTap: () {
    },
    child: Image.asset(
      '${info["file"]}.png',
      width: 100.0,
      height: 100.0,
    ),
  );
}
```
