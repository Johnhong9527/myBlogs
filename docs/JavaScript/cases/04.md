# 正则截取指定字符串

#### [参考文章](http://www.baidufe.com/item/eb10deb92f2c05ca32cf.html)
#### 正则截取指定字符串
```javascript
let str='{"asd":\\1234\\1234\\1234\\1234,"true"},
         {"sada":\\1111\\1234\\1234\\4444,"true"}';
let substr = str.match(/"sada":(\S*),"true"/);
console.log(substr[1]); // \\1111\\1234\\1234\\4444 
```
