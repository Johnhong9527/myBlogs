# L2TP Client之路

### 添加源
```bash
sudo add-apt-repository ppa:nm-l2tp/network-manager-l2tp
```
### 更新缓存
```bash
sudo apt-get update
```
### 安装network-manager-l2tp
```bash
sudo apt install network-manager-l2tp
```
### 安装UI
```bash
sudo apt install network-manager-l2tp-gnome
```
