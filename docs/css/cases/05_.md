# 垂直居中

```css
div {
    width: 50%;
    margin: 0 auto;
    height: 200px;
    line-height: 200px;
    background-color: antiquewhite;
    text-align: center;
}

div img {
    vertical-align: middle;
    display: inline-block;
    /*margin: 0 0 -20px 0;*/
}
```
