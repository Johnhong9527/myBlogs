FAQ Examples

安装mysql

# 检查系统是否已经安装mysql
which mysql  # /user/bin/mysql
# 安装mysql # 安装时根据安装提示设置相关mysql
sudo apt-get install mysql-server mysql-client
# 测试是否安装成功
sudo netstat -tap | grep mysql
# 登录mysql
mysql -u root -p
# 设置密码
mysqladmin -u root password "new_password";

登录

mysql -u 用户名 -p
Enter password: 输入密码
